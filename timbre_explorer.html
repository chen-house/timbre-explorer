音色是什么？
听见差异，理解原理

用耳朵听，用眼睛看。交互探索波形音色差异与音色分类体系。

波形 = 频谱结构 = 音色

四种基础波形，频谱结构不同，音色截然不同。点击卡片播放，用耳朵感受差异。

SINE WAVE
正弦波
Sine Wave
只含基频 无泛音 最纯净
频谱最简单，只有一个频率成分。听起来像"哔"的纯音，是所有复杂波形的基础单元。
SQUARE WAVE
方波
Square Wave
奇次谐波 空洞感 复古游戏感
只含奇次谐波（1、3、5…倍频）。有特殊的"空洞"质感，是早期游戏机、8-bit音乐的标志性音色。
SAW WAVE
锯齿波
Sawtooth Wave
所有谐波 最丰富 明亮/厚实
含所有谐波（奇次+偶次）。频谱最丰富，听起来最"厚"最"亮"。合成器铅音、Pad的首选起点。
TRIANGLE WAVE
三角波
Triangle Wave
奇次谐波 衰减更快 柔和
和方波一样只含奇次谐波，但高次谐波衰减更快，所以比方波更柔和，接近于笛子的纯净感。

时间包络：同样的波形，不同的音色

把钢琴音频反转——起音变成尾音，瞬态感变成弦乐渐入感。频谱没变，但音色感受完全不同。ADSR是塑造音色的时间维度。

CORE INSIGHT · 核心洞见
把钢琴音频反转播放：Attack 变成 Release，Release 变成 Attack。
频谱结构完全没有改变，但音色从"打击感"变成了"弦乐渐入感"。

这说明：时间包络对音色感知的影响，不亚于频谱本身。
同样的"材料"，不同的"时间形状"，就是不同的音色。
